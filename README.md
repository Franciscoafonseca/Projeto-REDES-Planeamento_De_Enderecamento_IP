# Projeto de Redes de Computadores - Packet Tracer

## Descrição

Este projeto foi desenvolvido no âmbito da unidade curricular de Redes de Computadores e consiste na criação e configuração de uma rede no Cisco Packet Tracer. O ficheiro principal do projeto encontra-se no formato `.pkt`, contendo a topologia de rede, os dispositivos utilizados e as respetivas configurações. 

O objetivo do trabalho é aplicar conceitos fundamentais de redes, incluindo planeamento da topologia, endereçamento IP, interligação entre dispositivos e validação da comunicação entre diferentes segmentos da rede.

## Objetivos

Os principais objetivos deste projeto são:

- construir uma topologia de rede funcional no Cisco Packet Tracer;
- configurar dispositivos de rede, como routers, switches e hosts;
- aplicar esquemas de endereçamento IP adequados;
- testar a conectividade entre máquinas da rede;
- consolidar conhecimentos práticos de configuração e simulação de redes.

## Ferramentas utilizadas

- Cisco Packet Tracer
- Conceitos de redes de computadores
- Configuração de dispositivos Cisco

## Conteúdo do projeto

O projeto é composto essencialmente por:

- ficheiro `.pkt` com a simulação da rede;
- definição da topologia;
- configuração dos dispositivos;
- testes de comunicação entre os vários elementos da rede.

## Possíveis elementos abordados

Dependendo da implementação realizada no ficheiro Packet Tracer, o projeto pode incluir conceitos como:

- endereçamento IPv4;
- sub-redes;
- configuração de interfaces;
- routing estático ou dinâmico;
- VLANs;
- DHCP;
- DNS;
- ACLs;
- NAT;
- testes de conectividade com `ping` e `traceroute`.

## Como abrir o projeto

1. Abrir o Cisco Packet Tracer.
2. Selecionar a opção para abrir um ficheiro existente.
3. Carregar o ficheiro `.pkt` incluído neste projeto.
4. Analisar a topologia e testar a comunicação entre dispositivos.

## Como testar

Depois de abrir o ficheiro no Packet Tracer, podem ser realizados vários testes para validar o funcionamento da rede, tais como:

- verificar as configurações IP dos hosts;
- confirmar o estado das interfaces dos routers e switches;
- testar conectividade com comandos `ping`;
- verificar se existe comunicação entre diferentes redes;
- validar regras ou serviços configurados.

## Estrutura do projeto

A estrutura do projeto é simples e centra-se no ficheiro de simulação da rede:

- `PROJ_RCD_2023_2024_1F_FRANCISCOAFONSECA_N2120622_VERONICAMEDINA_N2121022_PL2_PKT.pkt` - ficheiro principal do Cisco Packet Tracer com a implementação da rede.

## Requisitos

Para visualizar e testar este projeto é necessário:

- ter o Cisco Packet Tracer instalado;
- utilizar uma versão compatível com o ficheiro `.pkt`;
- possuir conhecimentos básicos de configuração e análise de redes.

## Possíveis melhorias

Algumas melhorias que podem ser adicionadas ou exploradas no projeto incluem:

- documentação detalhada da topologia;
- inclusão de uma tabela de endereçamento IP;
- explicação das configurações feitas em cada dispositivo;
- separação lógica da rede com VLANs;
- implementação de mecanismos adicionais de segurança;
- expansão da topologia com novos serviços ou sub-redes.

## Conclusão

Este projeto permite aplicar, em contexto prático, vários conceitos fundamentais de redes de computadores através da simulação no Cisco Packet Tracer. A construção e configuração da rede ajudam a consolidar competências relacionadas com planeamento, implementação, configuração e teste de infraestruturas de comunicação.
